# Asignatura DWES - Proyecto Tema 3
Proyecto que contiene todas las prácticas de PHP del Tema 3.

## Contenido.
* Uso de variables.
* Uso de la clase DateTime.
* Variables superglobales.
* Uso de librerías.
* Uso de arrays.
* Creación de formularios.

## Lenguajes utilizados:
* HTML
* CSS
* PHP 7.4
